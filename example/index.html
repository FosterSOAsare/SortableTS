SortableTS

TypeScript library for reorderable drag-and-drop lists

Features

  • Simple list
  • Shared lists
  • Cloning
  • Disable sorting
  • Handles
  • Filter

Examples

  • Grid
  • Nested sortables
  • MultiDrag
  • Swap

// Loop through each nested sortable element
            document.querySelectorAll('.nested-sortable').forEach(el => {
              new Sortable(el, {
                group: 'nested',
                animation: 150,
                fallbackOnBody: true,
                swapThreshold: 0.65
              });
